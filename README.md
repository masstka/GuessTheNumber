# Gra w zgadywanie liczby / Guess the Number Game

## Opis / Description

To prosta gra, w której komputer losuje liczbę z zakresu 1-100, a zadaniem gracza jest zgadnięcie tej liczby. Program informuje, czy podana liczba jest za mała, za duża, czy też gracz zgadł poprawnie.

---

This is a simple number guessing game where the computer randomly selects a number between 1 and 100. The player needs to guess the number. The game will inform the player if the guess is too small, too large, or correct.

## Jak uruchomić / How to Run

1. Pobierz lub sklonuj repozytorium. / / Download or clone the repository.
2. Uruchom plik w Pythonie. / Run the Python file.
3. Zgaduj liczbę! / Run the Python file and guess the number!
